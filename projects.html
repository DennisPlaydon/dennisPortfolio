GST calculator

A client spends hours doing his GST related tax every 3 months. I developed a piece of custom software that would loop through three months of statements and organize each transaction into a payment to the client or a transaction has the potential for tax to be claimed back.

I used two libaries to aid in development. I used openpyxl and re. Openpyxl allowed my program to read the spreadsheet and alter values. The re library is what I used to create regular expressions to pick up payments/transactions.

Easy Search

I often have the command prompt open while on my computer. I designed an easySearch script that would allow me to extend the functionality of the command prompt. There are a few main features I can use such as news, weather, maps, google and youtube.

I found the maps feature to be the most interesting to implement. The easySearch map command will take an address from your clipboard and take the user straight to the address on Google Maps.

Easy Search

Information Redactor

This is another script that takes a chunk of text the user has previously copied and extracts any visa credit card numbers out of it. It will then replace the last 12 numbers with stars and copy the credit card numbers to the user keyboard.

I used the pyperclip library to access the users clipboard. The program pulls the credit card numbers by using regular expressions to match credit card patterns.

Rugby Sort

This project involved sorting a list of rugby teams into a table ranking from the top team to the bottom. Multiple teams had similar scores or points so the program needed to consider a variety of different factors before determining the teams position.

It was interesting to implement my own search algorithm as it gave me some insight into the complexities of very efficient algorithms and the work that goes into creating them.

Java

Shape Spawner

This program allowed the user to interact with a GUI and click to spawn shapes which then moved on the screen following a path. The GUI had multiple options to change fill colour, size, shape, path and border colour.

I used an inheritance structure so that every shape spawned is an instance of the superclass. This was my first major java program where I learnt the principles of OOP, inheritance structures and using different data structures such as Arraylists.
